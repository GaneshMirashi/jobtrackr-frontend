# JobTrackr Frontend 🚀

JobTrackr is an AI-powered job application tracking system designed to help users manage their job search efficiently.

This repository contains the frontend application built with modern technologies, providing a clean UI, real-time updates, and an intuitive Kanban board experience.

## ✨ Features

- 🔐 JWT Authentication (Login/Register)
- 📊 Dashboard with analytics and charts
- 📌 Kanban board with drag-and-drop
- 🔍 Search, filter, and sort applications
- ⚡ Real-time UI updates with React Query
- 🎨 Clean and modern UI with Tailwind CSS

## 🛠 Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React Query (TanStack Query)
- Zustand (state management)
- Axios
- @dnd-kit (drag and drop)
- Recharts

## 🌐 Backend

This frontend connects to the JobTrackr backend API built with Django REST Framework.
